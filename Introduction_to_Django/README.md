# Introduction to Django

This directory is for your "Introduction_to_Django" assignment.

Quick start:
- From project root:
  - mkdir Introduction_to_Django
  - echo "# Introduction to Django" > Introduction_to_Django/README.md
- To start a Django project inside:
  - cd Introduction_to_Django
  - django-admin startproject mysite .
- Or to create an app:
  - django-admin startapp example_app

Commit the new folder so it's tracked by git:
git add Introduction_to_Django
git commit -m "Add Introduction_to_Django directory"
